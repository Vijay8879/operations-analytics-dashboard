# 💻 SQL Design

The dashboard query follows a modular approach using multiple Common Table Expressions (CTEs).

## Query Flow

```
Raw Orders
      │
      ▼
Daily Aggregation
      │
      ▼
Historical Trend
      │
      ▼
Forecast Allocation
      │
      ▼
KPI Calculation
      │
      ▼
Product Deactivation
      │
      ▼
Final Dashboard Output
```

## SQL Techniques Used

- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- Dynamic Filtering
- Conditional Aggregations
- KPI Calculations
- Forecast Allocation
